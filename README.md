## Run Locally

Clone the project

```bash
  git clone https://github.com/<yourusername>/stack-auth-rs
```

Go to the project directory

```bash
  cd stack-auth-rs
```

Build project

```bash
 cargo build
```

Start the server

```bash
  cargo run
